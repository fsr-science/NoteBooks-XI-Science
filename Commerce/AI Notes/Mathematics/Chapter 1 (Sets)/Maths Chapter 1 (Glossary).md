# Glossary: Chapter 1 - Sets

### **Core Definitions**
*   **Set:** A well-defined collection of objects [1, 2].
*   **Elements / Members:** Synonymous terms for the objects that belong to a set [3].
*   **Well-defined:** A collection is well-defined if one can definitely decide whether a particular object belongs to it or not [4].
*   **Empty Set (Null or Void Set):** A set which does not contain any element, denoted by the symbol $\phi$ or $\{ \}$ [2, 5].
*   **Finite Set:** A set that is empty or consists of a definite number of elements [2, 6].
*   **Infinite Set:** A set where the number of elements is not finite [2, 6].
*   **Equal Sets:** Two sets $A$ and $B$ that have exactly the same elements ($A = B$) [7, 8].
*   **Universal Set ($U$):** A basic set containing all objects relevant to a particular context, from which other sets are formed as subsets [9, 10].

### **Representations**
*   **Roster (Tabular) Form:** A method of representing a set where all elements are listed, separated by commas, and enclosed within braces $\{ \}$ [11].
*   **Set-builder Form:** A method where all elements of a set are described by a single common property [12, 13].

### **Subsets and Intervals**
*   **Subset:** Set $A$ is a subset of set $B$ ($A \subset B$) if every element of $A$ is also an element of $B$ [8, 14].
*   **Proper Subset:** If $A \subset B$ and $A \neq B$, then $A$ is called a proper subset [15].
*   **Superset:** If $A$ is a proper subset of $B$, then $B$ is called the superset of $A$ [15].
*   **Singleton Set:** A set that contains only one single element [15].
*   **Interval (Open):** The set of real numbers $\{y : a < y < b\}$, denoted by $(a, b)$, where end points are excluded [16, 17].
*   **Interval (Closed):** The set of real numbers $\{x : a \leq x \leq b\}$, denoted by $[a, b]$, where end points are included [17].

### **Set Operations**
*   **Union ($A \cup B$):** The set consisting of all elements that are in either $A$ or $B$ (common elements are taken only once) [8, 18, 19].
*   **Intersection ($A \cap B$):** The set of all elements which are common to both $A$ and $B$ [8, 20, 21].
*   **Difference ($A - B$):** The set of elements which belong to $A$ but do not belong to $B$ [8, 22, 23].
*   **Complement ($A'$):** The set of all elements in the universal set $U$ that are not elements of $A$ ($A' = U - A$) [8, 24].
*   **Disjoint Sets:** Two sets $A$ and $B$ are disjoint if they have no common elements ($A \cap B = \phi$) [25].

### **Visual and Symbolic Tools**
*   **Venn Diagram:** A diagram using rectangles (for the universal set) and closed curves/circles (for subsets) to represent relationships between sets [26, 27].
*   **$\in$ (Epsilon):** Symbol denoting "belongs to" [3].
*   **$\subset$:** Symbol denoting "is a subset of" [28].
*   **$\Rightarrow$:** Symbol meaning "implies" [14].
*   **$\iff$:** Symbol for "if and only if" (iff) [29].

### **Standard Mathematical Sets**
*   **$\mathbb{N}$:** Set of all natural numbers [4].
*   **$\mathbb{Z}$:** Set of all integers [4].
*   **$\mathbb{Q}$:** Set of all rational numbers [4, 30].
*   **$\mathbb{R}$:** Set of all real numbers [4].
*   **$\mathbb{T}$:** Set of all irrational numbers [16].
